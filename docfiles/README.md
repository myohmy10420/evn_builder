# dotfiles

Everything will install by Homebrew and work on Linux or macOS system.

If in Linux, Homebrew requirements:
- GCC 4.7.0 or newer
- Linux 2.6.32 or newer
- Glibc 2.13 or newer
- 64-bit x86_64 CPU


## Installation

    git clone https://github.com/myohmy10420/dotfiles ~/dotfiles
    cd ~/dotfiles
    chmod 760 setup.sh
    ./setup.sh
